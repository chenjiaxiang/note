动词的　て変　た変　  
一类动词（五段动词）

|动词结尾|て变|原形举例|て变举例|た变举例|
|-|-|-|-|-|
|す|して|話す|話して|話した|
|く|いて|磨く|研いて|磨いた|
|ぐ|いで|急ぐ|急いで|急いた|
|う　つ　る|って|洗う|洗って|洗った|
|　|　|待つ|待って|待った|
|　|　|やる|やって|やった|
|ぶ　む　ぬ|んで|遊ぶ|遊んで|遊んた|
|　|　|読む|読んで|読んた|
|　|　|死ぬ|死んで|死んた|
|行く|行って|特例|||

特殊记忆法：  
る　う　つ　　撸五次  
ぬ　む　ぶ　　nmb
